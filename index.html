Lesson 01

Jump Ball

In this lesson, I familiarized myself with the interface, tools, and workflow in Rive. I applied the basic techniques in a hands-on project to understand the fundamentals of working with Rive and build a solid foundation in motion design!

Lesson 02

Water Bubble

In this lesson, I learned how to create realistic water effects. I discovered how to incorporate sliders to improve user interaction, and I applied these skills to simulate the natural, dynamic movement of water.

Lesson 03

Belly Mixing

In this lesson, I configured the character rigging and joystick controls. I refined the animation controls, learned techniques for smooth transitions between states to create seamless movements, and learned how to create more complex and realistic character movements.
